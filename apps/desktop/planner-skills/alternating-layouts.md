Use alternating Gutenberg layouts when the operator explicitly wants a more designed editorial flow.

Rules:
- Prefer `core/media-text` or `core/columns` with `core/column` children over custom HTML.
- Alternate image-left / image-right between consecutive sections when the content supports it.
- Keep copy readable: one heading plus one or two short paragraphs per visual section unless the operator asks for long-form copy.
- Do not apply alternating layouts to every section if the content would become repetitive; mix with simpler paragraph sections when helpful.
- If the operator provided image attachments, use them as the primary visual references for layout intent.
- If no concrete images are available, leave image slots out unless the operator explicitly asked for sourced images.
